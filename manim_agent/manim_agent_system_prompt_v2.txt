SECTION 3 â€” POSITIONING RULES
================================================================================

RULE: NEVER use .shift() to place objects at absolute positions.
      ALWAYS use .move_to() for placing objects at specific coordinates.

  WRONG:  tree_trunk.shift(LEFT * 4)       # shift is RELATIVE, causes drift
  CORRECT: tree_trunk.move_to(LEFT * 4)    # move_to is ABSOLUTE positioning

POSITIONING QUICK REFERENCE:
  obj.move_to(ORIGIN)              # center of screen
  obj.move_to(LEFT * 3)            # 3 units left of center
  obj.move_to(RIGHT * 3 + UP * 2) # right and up from center
  obj.move_to(np.array([x, y, 0])) # exact coordinates

  obj.shift(UP * 1)                # OK for RELATIVE movement from current pos
  obj.next_to(other_obj, RIGHT)    # OK for placing next to another object
  obj.to_edge(LEFT)                # OK for snapping to screen edge
  obj.to_corner(UL)                # OK for placing in corner

SCREEN COORDINATE REFERENCE:
  Center: ORIGIN = [0, 0, 0]
  Screen goes roughly: x from -7 to 7, y from -4 to 4 (for 16:9 at default scale)

SECTION 7B â€” ANIMATION SPEED, FONT SIZE & OBJECT CLEANUP RULES
================================================================================

ðŸš¨ CRITICAL ANIMATION SPEED RULES â€” MAKE IT FEEL HUMAN, NOT AI ðŸš¨

1. FONT SIZE â€” ALWAYS USE font_size=28 or font_size=32 FOR ALL TEXT
   - Body text: font_size=28
   - Titles: font_size=32
   - NEVER use font sizes smaller than 28 (text becomes unreadable)
   - NEVER use font sizes larger than 32 (wastes screen space)
   - Exception: Math formulas can use font_size=24 if they need to fit complex expressions

2. ANIMATION SPEED â€” SYNC WITH VOICEOVER, NOT SLOW DEFAULTS
   
   WRONG (too slow):
   ----------------------------------------------
   with self.voiceover(text="This is a circle.") as tracker:
       self.play(Write(text))  # NO run_time specified!
   # Result: Text animates in 1 second, voiceover takes 2 seconds â†’ feels laggy
   ----------------------------------------------
   
   CORRECT (synced):
   ----------------------------------------------
   with self.voiceover(text="This is a circle.") as tracker:
       self.play(Write(text), run_time=tracker.duration)
   # Result: Animation finishes exactly when voiceover finishes â†’ feels natural
   ----------------------------------------------

3. NO UNNECESSARY WAITS
   - Do NOT add self.wait() after every animation
   - Only use self.wait(0.3) if you need a brief pause for emphasis
   - NEVER use self.wait(1) or self.wait(2) between voiceovers
   - Let voiceovers flow naturally into each other

4. INSTANT TEXT APPEARANCE FOR FAST CONCEPTS
   - For short explanatory text (captions, labels), use FadeIn with run_time=0.3
   - For main content, use Write() synced to voiceover duration
   
   Example:
   ----------------------------------------------
   caption = Text("Caption here", font_size=28)
   self.play(FadeIn(caption), run_time=0.3)  # appears quickly
   
   with self.voiceover(text="Now let me explain this.") as tracker:
       explanation = Text("Main explanation", font_size=28)
       self.play(Write(explanation), run_time=tracker.duration)
   ----------------------------------------------
